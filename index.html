build

lib/build.doc.js

Build Benthos artifacts for AWS Lambda.

build(parameters: Object): Array<string>
Parameters
parameters (Object)
Name Description
parameters.configPath string (default package.json) Path to JSON file with a blobpack config key.
parameters.tmpRoot string (default tmp) Path to a temporary working directory.
parameters.configRoot string (default config) Path to the directory containing the artifact configs.
parameters.resourcesRoot string (default ./) Path to the directory all resources are relative to
parameters.distRoot string (default dist) Path to the directory to output artifacts
parameters.logger string (default console) Console compatible logger.
Returns
Array<string>: Generated artifacts paths.
